## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-4-03-01 | Elevation of Privilege | ActionPanel "Generate Invoice" button | mitigate | Button is hidden for non-compliance roles (frontend defense-in-depth); backend route enforces assertRole — frontend hiding alone is not the security control |
| T-4-03-02 | Tampering | noOfActivities form input | mitigate | parseInt(..., 10) prevents float injection; server action should validate noOfActivities is a positive integer when showNoOfActivities would be true; null for flat_fee/per_event is enforced in form logic |
| T-4-03-03 | Information Disclosure | invoiceStorageUrl passed as prop | accept | URL comes from server-side DB query (not user input); public R2 URL is intentional design (D-12); prop is read-only |
| T-4-03-04 | Repudiation | Invoice generation — who generated it | mitigate | Route handler stores generatedByClerkId and generatedByName in Invoice record; DB record is write-once (no update path) |
</threat_model>
